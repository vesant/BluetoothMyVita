# PSVita Bluetooth MAC Viewer

App simples para mostrar o endereco MAC Bluetooth da PSVita.

## Build (VitaSDK)
```sh
make
```

## Nota
O codigo inclui um stub para ler o MAC Bluetooth. Se a tua instalacao do VitaSDK expuser o API de Bluetooth, podemos ligar a chamada correta. Caso contrario, diz-me qual o header/funcao disponivel e eu ajusto.
